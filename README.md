## Beepbop Libraries

A collection of reusable packages for Beepbop projects.

### Packages

- **[bldr](packages/bldr/)** - TypeScript build tool with watch mode and automatic dist synchronization
- **[convex-cache](packages/convex-cache/)** - A library for caching Convex queries on the client and server
- **[convex-helpers](packages/convex-helpers/)** - A lightweight utility library for Convex containing helpers
- **[deep-equal](packages/deep-equal/)** - A tiny utility for deep structural equality
- **[deep-sort](packages/deep-sort/)** - A tiny utility that deeply normalizes JavaScript values
- **[local-db](packages/local-db/)** - A tiny client-side React hook for persistent state, backed by IndexedDB
- **[next-router](packages/next-router/)** - Type-safe route and search params for Next.js App Router with Zod validation
- **[pkce](packages/pkce/)** - Zero-dependency PKCE library with pure JavaScript SHA-256 implementation
- **[tsconfigs](packages/tsconfigs/)** - Shared TypeScript configuration files for projects
- **[zod-env](packages/zod-env/)** - Type-safe environment variable validation using Zod

### Usage

These packages are open sourced and can be used by anyone for their own projects.
